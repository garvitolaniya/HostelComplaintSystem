# 🏨 Hostel Complaint Management System (Java)

An enhanced console-based Java application designed to efficiently manage hostel complaints with advanced features like priority handling, categorization, search functionality, and statistics tracking.

---

## 🚀 Problem Statement

In hostels, complaints related to electricity, water, WiFi, and mess are often not managed efficiently. Lack of proper tracking leads to delays and unresolved issues.

---

## 💡 Solution

This project provides a structured system where:
- Students can register complaints with priority and category
- Complaints are tracked using unique IDs
- Admin can resolve and manage complaints
- Users can search and analyze complaint data

---

## ⚙️ Features

### 🔹 Core Features
- ✅ Add new complaints
- 📋 View all complaints
- 🔄 Update complaint status (Pending → Resolved)

### 🔹 Advanced Features
- 🚨 Priority System (High / Medium / Low)
- 🗂️ Category System (Electrical / WiFi / Water / Mess / Other)
- 🔍 Search complaint by ID
- 📊 Statistics dashboard (Total / Resolved / Pending)
