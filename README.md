# 🏨 Hostel Complaint Management System (Java)

A simple console-based Java application designed to manage hostel complaints efficiently. This system allows students to register complaints and enables administrators to track and resolve them.

---

## 🚀 Problem Statement

In hostels, students often face issues like water shortage, electricity problems, WiFi issues, and maintenance delays. These complaints are usually not tracked properly, leading to delays and inefficiencies.

---

## 💡 Solution

This project provides a centralized system where:
- Students can register complaints
- Complaints are stored and tracked
- Admin can mark complaints as resolved

---

## ⚙️ Features

- ✅ Add new complaints
- 📋 View all complaints
- 🔄 Update complaint status (Pending → Resolved)
- 🆔 Unique ID for each complaint
- 📊 Simple and user-friendly console interface
